## Installation
- WebVisu.html mittels eines beliebigen FTP-Clients (z.B. WinSCP oder Filezilla) auf die Steuerung unter /webserv kopieren.

## Aufrufen

http://[IpMeinerSteuerung]/webserv/WebVisu.html
